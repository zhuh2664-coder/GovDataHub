# GovDataHub
GovDataHub – A lightweight, extensible data catalog system for government data governance. Enables cross-departmental metadata management, semantic search, and data discovery — built for real-world policy and interoperability needs. Starting from a minimal prototype, evolving toward a full-stack solution aligned with DCMM and DAMA-DMBOK.
GovDataHub 是一个面向政务场景的轻量级、可扩展数据目录系统，旨在解决政府数据“找不到、看不懂、不敢用”的核心痛点。项目从最小可行原型起步，逐步构建支持元数据管理、跨域语义检索、数据血缘追踪与质量评估的能力，并与 DCMM（数据管理能力成熟度模型）和 DAMA-DMBOK 治理框架对齐。
